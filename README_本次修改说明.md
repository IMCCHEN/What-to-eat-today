# 本次修改说明

本版基于“更新表格后的 v2 替换包”继续修改，主要修正“现在准备吃 / 营业时间”相关逻辑。

## 修改内容

1. 删除“早餐”选项。
   - 状态推荐页的“现在准备吃”不再出现早餐。
   - 随机推荐自动判断时间时，也不会再返回 breakfast。

2. 统一三类正餐时间段：
   - 午餐：11:00–13:30
   - 晚餐：17:30–20:00
   - 夜宵：21:00 及以后

3. 推荐时加入营业时间匹配：
   - 若菜品/餐厅的 `businessHours` 能解析出营业区间，则按上面的时间段进行匹配。
   - 若没有明确营业时间，但有 `timeTag`，则用 `午餐 / 晚餐 / 夜宵 / 下午茶` 标签兜底判断。
   - 若营业时间和标签都没写，则不强制排除，但结果中会提示“营业时间未填写，需现场确认”。

4. 保留之前功能：
   - 米饭/面条/饮品单选硬筛选；多选按多选范围推荐。
   - 食堂小菜自动组合为“主食 + 菜品”的堂食方案。
   - 减脂中支持少主食/无明显主食的高蛋白推荐。
   - 商圈推荐展示商家名。
   - “肉多多！！”偏好继续生效。
   - 个人中心补充菜品/餐厅交互继续保留。

## 使用方式

把本文件夹内同名文件覆盖到原网站项目中即可。更新 GitHub 后，原 GitHub Pages 链接不变；打开网页后可按 Ctrl + F5 强制刷新。
